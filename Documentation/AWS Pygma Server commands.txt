Connect to EC2:
    ssh ec2-user@16.171.145.64

Install Docker:
    sudo su -
    yum update
    yum search docker
    yum info docker
    yum install -y docker
    yum install -y npm

Create Docker group to don't use sudo:
    usermod -a -G docker ec2-user
    id ec2-user
    newgrp docker

Install docker-compose:
    sudo curl -L https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m) -o /usr/local/bin/docker-compose
    sudo chmod +x /usr/local/bin/docker-compose

Enable Docker service:
    systemctl enable docker.service

Start the Docker service
    systemctl start docker.service

Make docker auto-start
    sudo chkconfig docker on

Copy the respective target folders from backend projects and the .next from the frontend project

Run the commands from 'Docker AWS Pygma EC2 commands.txt'
